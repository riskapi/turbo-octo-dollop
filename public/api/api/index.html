1. GET /v1/echo Link to heading

Description:
The Echo service returns the request details sent by the client, including the HTTP method, headers, query parameters, and body (if present). This is useful for testing and debugging API requests.

Endpoint:
GET /v1/echo

Request Example:

curl -X GET "https://api.cameronmichelis.com/v1/echo?param=value"
